# Screenshots Auto Wize

Mocks gerados para referência visual (proporção 9:16 / ~6.7").

**Antes de enviar à App Store**, substitua por capturas reais do Simulator:

1. Xcode → Simulator → iPhone 16 Pro Max (6.7") e um modelo 6.5"
2. Percorra: Login → Home → Entrega → Troca
3. Device → Screenshots (⌘S)
4. Se oferecer iPad, capture também no iPad Pro 13"
